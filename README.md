# Cloud-Native Microservice Application

This project demonstrates a simple cloud-native application built using Python, Docker, Kubernetes, and CI/CD pipelines.

## 🚀 Features
- REST API built with Flask
- Containerized using Docker
- Kubernetes deployment configuration
- CI/CD pipeline using GitHub Actions

## 🏗 Architecture
User → Flask App → Docker Container → Kubernetes Deployment

## 🛠 Tech Stack
- Python (Flask)
- Docker
- Kubernetes
- GitHub Actions (CI/CD)

## ▶️ Run Locally
```bash
pip install -r requirements.txt
python app.py
